# 📊 Financial Analysis Signals

Este proyecto contiene herramientas y scripts en Python para consultar datos de criptomonedas en tiempo real, aplicar indicadores técnicos y generar señales de trading de forma automatizada.

## 📁 Estructura del Proyecto

- `ANALIZADOR_CRIPTO.py`: Consulta de precios, indicadores técnicos y generación de señales tipo semáforo (RSI, MACD, etc.).
- `utils/`: Funciones auxiliares para APIs, formateo de datos, etc.
- Otras carpetas o archivos específicos para pruebas, gráficos o integración con Streamlit.

## ⚙️ Tecnologías usadas

- Python 3.+
- Pandas, NumPy, Requests
- TA / Pandas_TA para indicadores técnicos
- Colorama y Tabulate para salida en consola
- Matplotlib para gráficos (opcional)
